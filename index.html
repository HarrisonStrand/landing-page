    Week 1 Projects

  1. My First Webpage

    2. This was my first attempt at creating a webpage using HTML and a little bit of CSS. I created live links to navigate to my favorite things and places, as well as to navigate to the Epicodus home page.

  2. SPRANKLZ

    3. This was a very fun collaborative project making a mock cupcake website. We used HTML to write a description and title of our website, and then CSS to style the color of text and align all of our cupcake picture examples.

  3. Animal Shelter

    4. In this project, my partners and I created a mock pet adoption site with exotic animals and a little back story about them using HTML and CSS. We learned about floating and different types of alignment to section each animal in a list with a corresponding picture next to the description.

  4. Recreating Wikipedia

    5. This project was a major challenge for us new devs. We tried our best to recreate the Wikipedia homepage with HTML, CSS and Bootstrap. We created and aligned a navigation bar as well as specified links for our top nav row. It was very interesting comparing code and seeing how complicated certain professional and stable proffesional websites can be.

  5. Bootstrap Practice

    6. When the week was almost done, we followed along in a lesson to see all of the amazing things that bootstrap can do to style and polish your HTML file. This was a great example of trying out different bootstrap alerts and colors, as well as different ways to contain elements in your project.
